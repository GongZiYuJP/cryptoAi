# ETH AI交易机器人使用说明

## 功能特点

✅ **实时监控ETH走势** - 每5分钟自动分析一次ETH价格走势  
✅ **多技术指标分析** - 使用MA、EMA、RSI、MACD、VOL、布林带等指标  
✅ **智能信号生成** - 综合多个指标评分，只发送高胜率信号  
✅ **Telegram通知** - 发现交易机会时自动发送详细通知  
✅ **详细交易建议** - 包含入场价格、止盈、止损、盈亏比等信息  

## 技术指标说明

### 1. MA/EMA (移动平均线)
- **MA7, MA14, MA21, MA50, MA100, MA200** - 多个周期的简单移动平均线
- **EMA7, EMA14, EMA21, EMA50, EMA100, EMA200** - 多个周期的指数移动平均线
- 用于判断趋势方向和支撑/阻力位

### 2. RSI (相对强弱指标)
- 范围：0-100
- **< 30**: 超卖区域，可能反弹（看多信号）
- **> 70**: 超买区域，可能回调（看空信号）
- **30-70**: 中性区域

### 3. MACD (指数平滑异同移动平均线)
- **MACD线**: 快慢均线差值
- **信号线**: MACD的移动平均
- **MACD柱**: MACD与信号线的差值
- **金叉**: MACD上穿信号线，看多
- **死叉**: MACD下穿信号线，看空

### 4. VOL (成交量分析)
- **成交量比率**: 当前成交量 / 20日均量
- **> 1.5**: 放量，配合价格上涨/下跌有重要意义
- **< 0.7**: 缩量，市场观望

### 5. 布林带 (Bollinger Bands)
- **上轨**: 价格上限
- **中轨**: 20日均线
- **下轨**: 价格下限
- 价格接近下轨可能反弹，接近上轨可能回调

## 信号评分系统

机器人使用综合评分系统（0-100分）：

- **RSI分析**: 20分
- **MA/EMA均线**: 25分
- **MACD**: 25分
- **成交量**: 15分
- **布林带**: 15分

**信号阈值**: 75分（可调整 `SIGNAL_THRESHOLD`）

只有当综合得分 ≥ 75分时，才会发送Telegram通知。

## 使用方法

### 1. 启动实时监控（推荐）

```bash
python cryptoAiBot.py
```

机器人将：
- 立即分析一次ETH
- 每5分钟自动检查一次
- 发现强信号时发送Telegram通知

### 2. 立即分析ETH

```bash
python cryptoAiBot.py --analyze-eth
```

立即分析一次ETH走势，如果有强信号会发送通知。

### 3. 分析所有币种

```bash
python cryptoAiBot.py --analyze-all
```

分析BTC、ETH、XRP、SOL、BNB的走势，发送市场概览。

### 4. 测试Telegram连接

```bash
python cryptoAiBot.py --test-telegram
```

测试Telegram Bot配置是否正确。

### 5. 查看账户状态

```bash
python cryptoAiBot.py --check
```

查看当前账户余额和持仓情况。

## 配置参数

在 `cryptoAiBot.py` 中可以调整以下参数：

```python
# 信号阈值（0-100）
SIGNAL_THRESHOLD = 75  # 提高此值会减少信号数量，但信号质量更高

# 止损止盈
STOP_LOSS_PCT = 0.02   # 2%止损
TAKE_PROFIT_PCT = 0.06 # 6%止盈（盈亏比3:1）

# 监控间隔（秒）
MONITOR_INTERVAL = 300  # 5分钟

# 时间周期
TIMEFRAME = '1h'  # 可选: '1m', '5m', '15m', '1h', '4h', '1d'
```

## 常见问题

**Q: 为什么没有收到通知？**  
A: 检查以下几点：
- 信号强度是否达到阈值（默认75分）
- Telegram Bot Token和Chat ID是否正确
- 是否已向Bot发送过至少一条消息
- 运行 `--test-telegram` 测试连接

**Q: 如何调整信号敏感度？**  
A: 修改 `SIGNAL_THRESHOLD` 参数：
- 降低（如60）：更多信号，但可能包含弱信号
- 提高（如85）：更少信号，但信号质量更高

**Q: 可以同时监控多个币种吗？**  
A: 当前版本专注于ETH，但可以修改代码添加其他币种的监控。

**Q: 机器人会自动下单吗？**  
A: 不会。当前版本只发送分析信号，不会自动下单。您可以根据信号手动操作。
